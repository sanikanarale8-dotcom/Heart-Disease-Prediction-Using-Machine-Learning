# Heart Disease Prediction Using Machine Learning

## Project Overview

This project uses Machine Learning techniques to predict whether a patient is likely to have heart disease based on medical attributes such as age, cholesterol level, blood pressure, chest pain type, and other health indicators.

The project was developed as part of the Summer Internship 2026.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

---

## Dataset Information

* Total Records: 303
* Total Features: 14
* Target Variable: Heart Disease Status

Key Features:

* Age
* Sex
* Chest Pain Type
* Cholesterol
* Blood Pressure
* Maximum Heart Rate
* Fasting Blood Sugar
* Target

---

## Machine Learning Models Used

### Logistic Regression

Accuracy: 86.89%

### Random Forest

Accuracy: 83.61%

---

## Results

The Logistic Regression model achieved the highest accuracy of 86.89% and was selected as the final prediction model.

The system successfully predicts whether a patient has heart disease based on medical information.
